# Changelog

## rpathviz 0.1.1

- Added `low_tl_spread` parameter to
  [`webplotviz()`](https://biadias.github.io/rpathviz/reference/webplotviz.md).
  Nodes within the same rounded trophic-level band are now re-spaced by
  rank, guaranteeing visible separation at low trophic levels regardless
  of the KK layout positions. Default (`low_tl_spread = 1`).

## rpathviz 0.1.0

- Initial submission and package release.
